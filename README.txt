3-axis Robot control in Python for Windows
=======================================

This program is based on Arbotix software to access the robot controller.  There are some include files that are needed to have this 
program work, pypose.py and the driver.py are some of the files that come with the Arbotix software (freeware).

This program will rotate and make some 3-axis movement to the robot at selected speed and selected movement using the input
commands given from the python script.
